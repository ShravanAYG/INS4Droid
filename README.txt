INS4Droid

Inertial navigation system using built in androind sensosors.
Super fast and light weight built with C/SDL2.
Track your location without GNSS, GSM etc just by using a initial frame of reference.

Features:
- No network connection requires
- Average deviation less than 1 m per 2 km (real-world tests)
- Super fast

How to build:
This project can be built by importing android-project in android studio or by using android studio cli tools
